# python-file-organizer
A python script that automatically organizes files in a folder 
into categories such as Images, Videos , Documents , and Music.

# Features
- Organizes files by type
- Supports Images , Videos , Documents , Music ,  and others
- works on Windows and Mac

# How to Use
1.Run `file_organizer.py`
2.Enter the folder path you want to organize

# Technologies
- Python
- os module
- shutil module
